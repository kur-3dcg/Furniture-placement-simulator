# ブルーアーカイブ家具シミュレーションツール

Pixi.jsを使用したWebブラウザ上で動作する家具配置シミュレーションツールです。クォータービュー（アイソメトリック投影）で20x20のグリッド上に家具を自由に配置できます。

## 特徴

- 🎨 **直感的な操作**: マウスクリックで簡単に家具を配置・削除
- 🔄 **回転機能**: 家具を90度単位で回転可能
- 👁️ **リアルタイムプレビュー**: 配置前に配置可能な位置を視覚的に確認
- 🎯 **柔軟な家具管理**: 任意のサイズと名前で家具を追加可能
- 🖼️ **クォータービュー**: 立体的で見やすいアイソメトリック表示
- 🎪 **ビュー移動**: 中クリックドラッグでキャンバスを自由に移動

## デモ

ブラウザでHTMLファイルを開くだけで、すぐに使用できます。

## 必要要件

- モダンなWebブラウザ（Chrome, Firefox, Safari, Edge等）
- JavaScript有効化
- インターネット接続（Pixi.jsをCDNから読み込むため）

## 使い方

### 基本操作

| 操作 | 説明 |
|------|------|
| **左クリック** | 選択中の家具をグリッドに配置 |
| **右クリック** | クリックした位置の家具を削除 |
| **中クリック+ドラッグ** | ビュー全体を移動 |
| **Rキー** | 選択中の家具を90度回転 |
| **マウスホバー** | 配置プレビューを表示（緑=配置可能、赤=配置不可） |

### 家具の管理

#### 家具の追加

1. 右側パネルの「家具名」入力欄に名前を入力（例：ソファ、ベッド）
2. 幅と高さを数値で指定（1〜10マス）
3. 「家具を追加」ボタンをクリック
4. 家具リストに新しい家具が追加されます

#### 家具の選択

- 家具リストから配置したい家具をクリックして選択
- 選択された家具は緑色でハイライト表示されます

#### 家具の削除

- 各家具の右側にある「削除」ボタンをクリック
- 削除すると、その種類の家具が配置済みのものも含めて全て削除されます

### 配置のルール

- 家具同士を重ねて配置することはできません
- グリッドの範囲外には配置できません
- 配置プレビューで緑色に表示されている場合のみ配置可能です

## 技術仕様

### 使用技術

- **Pixi.js 7.3.2**: WebGLベースの2D描画ライブラリ
- **HTML5 Canvas**: レンダリング
- **JavaScript (ES6+)**: アプリケーションロジック

### グリッド仕様

- **サイズ**: 20x20マス
- **タイル幅**: 32ピクセル
- **タイル高さ**: 16ピクセル
- **投影方式**: アイソメトリック（クォータービュー）

### 視覚デザイン

- **家具の境界線**: 太線（3px）で明確に区別
- **グリッド線**: 細線（1px）で控えめに表示
- **家具の色**: 16色のカラーパレットから自動割り当て
- **プレビュー表示**: 半透明で配置予定位置を表示

## カスタマイズ

コード内の定数を変更することで、以下のカスタマイズが可能です：

```javascript
const GRID_SIZE = 20;        // グリッドサイズ
const TILE_WIDTH = 32;       // タイル幅
const TILE_HEIGHT = 16;      // タイル高さ
const LINE_THICK = 3;        // 家具境界線の太さ
const LINE_THIN = 1;         // グリッド線の太さ
```

カラーパレットは`colors`配列で定義されており、自由に色を追加・変更できます。

## ライセンス

このツールは教育・学習目的で作成されています。

## 注意事項

- データの永続化機能はありません（ページをリロードすると配置がリセットされます）
- オフライン環境では動作しません（Pixi.jsをCDN経由で読み込むため）
- 大量の家具を配置するとパフォーマンスが低下する可能性があります
